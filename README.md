# CoolDiag AI

Assistente HVAC per diagnosi climatizzatori.

## Versione
v0.5 alpha

## Moduli
- Diagnosi climatizzatori split e multisplit
- R32, R410A, R290, R454B
- Misure frigorifere
- Controlli visivi
- Esito con affidabilità
- Toolbox HVAC
- Libreria tecnica
- Storico locale
- Report stampabile/PDF

## Pubblicazione Netlify
1. Collegare questo repository a Netlify
2. Deploy directory: root
3. Build command: lasciare vuoto
4. Publish directory: lasciare vuoto oppure `/`
